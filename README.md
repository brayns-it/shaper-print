# Shaper Print
Shaper Print is a Printer Server for Shaper, based on ASP.NET only for Windows for 
RDLC rendering.

# Documentation
View the online documentation [https://kb.brayns.it/shelves/shaper](https://kb.brayns.it/shelves/shaper)
